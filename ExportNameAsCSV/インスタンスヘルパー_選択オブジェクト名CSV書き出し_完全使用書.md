# インスタンスヘルパー：選択オブジェクト名CSV書き出し 完全使用書

## 1. このアドオンの目的

このBlenderアドオンは、**現在選択しているオブジェクトの名前をCSVファイルとして書き出す**ためのツールです。

大量のオブジェクトを扱う作業で、次のような用途に使えます。

- 選択したオブジェクト名の一覧を外部に保存する
- Excel、Googleスプレッドシート、Notionなどでオブジェクト名を管理する
- インスタンス、プロキシ、配置物、アセットなどの名前リストを確認する
- 後工程で使うために、選択オブジェクトの名前だけをCSV化する
- チームメンバーに「どのオブジェクトを対象にしたか」を共有する

このアドオンは、オブジェクトそのものを変換・複製・削除するものではありません。  
**選択中のオブジェクト名だけをCSVに保存する安全な書き出しツール**です。

---

## 2. この使用書で扱うファイル

このセットでは、次の2つのファイルを使用します。

| ファイル名 | 用途 |
|---|---|
| `インスタンスヘルパー_選択オブジェクト名CSV書き出し_修正版.py` | Blenderにインストールするアドオン本体 |
| `インスタンスヘルパー_選択オブジェクト名CSV書き出し_完全使用書.md` | この使用書。使い方・メニュー説明・注意点をまとめたMarkdownファイル |

後から見返したときは、**「修正版.py」がBlenderに入れるファイル**、**「完全使用書.md」が読むファイル**です。

---

## 3. 元のアドオンから修正・整理した点

修正版では、初めて使う人にも分かりやすくするために、以下を整理しています。

- アドオン名を `Instance Helper - Selected Object Names CSV Exporter` に変更
- サイドバーのタブ名を `Instance Helper` に整理
- パネル名を `Object Name CSV Export` に整理
- 初期保存ファイル名を `instance_helper_selected_object_names.csv` に変更
- 選択数をパネル上に表示
- CSVのヘッダー行を出すかどうか選べる設定を追加
- 名前順に並べて書き出す設定を追加
- 保存先が空の場合のエラー表示を追加
- 保存フォルダが存在しない場合の作成処理を安全化
- CSV書き出し失敗時にエラー内容を表示
- 書き出し完了時に件数と保存先を表示

## 4. インストール方法

### 4-1. Blenderを開く

Blenderを起動します。

### 4-2. アドオン設定を開く

上部メニューから次の順に開きます。

```text
Edit > Preferences > Add-ons
```

日本語UIの場合は、おおよそ次の場所です。

```text
編集 > プリファレンス > アドオン
```

### 4-3. アドオンをインストールする

`Install...` ボタンを押して、次のファイルを選択します。

```text
インスタンスヘルパー_選択オブジェクト名CSV書き出し_修正版.py
```

### 4-4. アドオンを有効化する

インストール後、アドオン一覧で次の名前を探してチェックを入れます。

```text
Instance Helper - Selected Object Names CSV Exporter
```

チェックが入れば使用できます。

---

## 5. アドオンの場所

このアドオンは、3Dビューの右側サイドバーに表示されます。

### 開き方

1. 3D Viewportを開く
2. キーボードの `N` キーを押す
3. 右側にサイドバーが表示される
4. `Instance Helper` タブを開く
5. `Object Name CSV Export` パネルを使う

表示場所は次の通りです。

```text
3D Viewport > Sidebar > Instance Helper > Object Name CSV Export
```

## 6. メニュー・項目の説明

### 6-1. `Selected Object Names to CSV`

パネルの説明ラベルです。  
このパネルが「選択中のオブジェクト名をCSVに書き出すためのもの」であることを示します。

### 6-2. `Selected: 数字`

現在選択しているオブジェクト数を表示します。

例：

```text
Selected: 5
```

この場合、5個のオブジェクトが選択されています。  
この状態で書き出すと、基本的には5件の名前がCSVに出力されます。

### 6-3. `Save Path`

CSVファイルの保存先を指定する項目です。

初期状態では次のようになっています。

```text
//instance_helper_selected_object_names.csv
```

`//` は、現在開いている `.blend` ファイルを基準にした相対パスです。

例：

```text
//exports/object_names.csv
```

この場合、現在の `.blend` ファイルと同じ場所に `exports` フォルダを作り、その中に `object_names.csv` を保存します。

絶対パスも使用できます。

Windowsの例：

```text
C:\Users\UserName\Desktop\object_names.csv
```

macOSの例：

```text
/Users/UserName/Desktop/object_names.csv
```

拡張子 `.csv` を付け忘れた場合でも、自動で `.csv` が追加されます。

### 6-4. `Include Header`

CSVの1行目にヘッダーを入れるかどうかを指定します。

オンの場合：

```csv
name
Cube
Sphere
Camera
```

オフの場合：

```csv
Cube
Sphere
Camera
```

ExcelやGoogleスプレッドシートで管理する場合は、オンのままがおすすめです。

### 6-5. `Sort by Name`

オブジェクト名を名前順に並べて書き出すかどうかを指定します。

オンの場合：

```text
A-Z順に並べて出力
```

オフの場合：

```text
Blenderの選択オブジェクトリストの順番で出力
```

大量のオブジェクトを管理する場合は、オンにすると確認しやすくなります。

選択した流れに近い状態で出したい場合は、オフにしてください。

### 6-6. `Export Selected Names to CSV`

実際にCSVを書き出すボタンです。

このボタンを押すと、現在選択されているオブジェクトの名前が `Save Path` に指定したCSVファイルへ保存されます。

## 7. 基本的な使い方

### 手順1：書き出したいオブジェクトを選択する

3Dビューまたはアウトライナーで、CSVに書き出したいオブジェクトを選択します。

複数選択も可能です。

### 手順2：サイドバーを開く

3Dビュー上で `N` キーを押して、右側サイドバーを表示します。

### 手順3：`Instance Helper` タブを開く

右側サイドバーのタブから `Instance Helper` を選択します。

### 手順4：保存先を指定する

`Save Path` にCSVの保存先を入力します。

例：

```text
//exports/selected_names.csv
```

### 手順5：必要に応じてオプションを設定する

通常は次の設定で問題ありません。

```text
Include Header: オン
Sort by Name: 必要に応じてオン
```

### 手順6：書き出しボタンを押す

`Export Selected Names to CSV` を押します。

成功すると、Blender下部またはステータス表示に次のようなメッセージが出ます。

```text
CSV書き出し完了: 5件 / 保存先パス
```

### 手順7：CSVファイルを確認する

保存先にCSVファイルが作成されているか確認します。

CSVはExcel、Googleスプレッドシート、テキストエディタなどで開けます。

## 8. 出力されるCSVの例

`Include Header` がオンの場合、次のようなCSVになります。

```csv
name
SM_Wall_01
SM_Wall_02
SM_Window_A
SM_Door_A
BP_Instance_Proxy_001
```

`Include Header` がオフの場合、次のようになります。

```csv
SM_Wall_01
SM_Wall_02
SM_Window_A
SM_Door_A
BP_Instance_Proxy_001
```

## 9. 具体的な使用例

### 9-1. 選択したアセット名を一覧化したい場合

1. アウトライナーで対象オブジェクトを選択
2. `Save Path` を `//exports/asset_list.csv` に設定
3. `Include Header` をオン
4. `Sort by Name` をオン
5. `Export Selected Names to CSV` を押す

名前順に整理されたアセット名リストがCSVで出力されます。

### 9-2. 現在選択しているインスタンス候補を記録したい場合

1. シーン内で対象のインスタンス、プロキシ、配置物などを選択
2. `Save Path` を `//exports/instance_targets.csv` に設定
3. `Export Selected Names to CSV` を押す

後から「どのオブジェクトを対象にしたか」を確認できます。

### 9-3. チームに選択対象を共有したい場合

1. 対象オブジェクトを選択
2. CSVを書き出す
3. CSVを共有する

Blenderを直接開かなくても、対象オブジェクト名を確認できます。

## 10. 注意点

### 10-1. 書き出されるのは「オブジェクト名」のみ

このアドオンがCSVに出力するのは、選択中のオブジェクトの名前だけです。

次の情報は出力しません。

- 位置
- 回転
- スケール
- マテリアル
- メッシュデータ
- コレクション名
- カスタムプロパティ
- インスタンス元の情報

### 10-2. 同じファイル名に書き出すと上書きされる

同じ `Save Path` を指定して再度書き出すと、前のCSVは上書きされます。

残しておきたい場合は、ファイル名を変えてください。

例：

```text
//exports/selected_names_2026_05_19.csv
```

### 10-3. オブジェクトを選択していないと書き出せない

何も選択していない状態でボタンを押すと、警告が出て処理はキャンセルされます。

```text
選択オブジェクトがありません。書き出したいオブジェクトを選択してください。
```

### 10-4. `.blend` を保存していない場合の `//` パスに注意

`//` から始まるパスは、基本的に現在の `.blend` ファイルの場所を基準にします。

まだ `.blend` を保存していない場合、意図した場所に出力されないことがあります。  
安全に使う場合は、先に `.blend` を保存してからCSVを書き出してください。

## 11. トラブルシューティング

### Q1. `Instance Helper` タブが表示されません

次を確認してください。

- アドオンが有効化されているか
- 3D Viewportで `N` キーを押しているか
- 右側サイドバーのタブを横にスクロールしていないか
- Blenderを再起動しても表示されるか

### Q2. CSVが作成されません

次を確認してください。

- オブジェクトを選択しているか
- `Save Path` が空になっていないか
- 保存先フォルダに書き込み権限があるか
- 保存先パスに使用できない文字が入っていないか
- すでに同名CSVをExcelなどで開いたままにしていないか

### Q3. 日本語名のオブジェクトが文字化けします

この修正版は `utf-8-sig` でCSVを書き出します。  
Excelでも日本語が比較的文字化けしにくい形式です。

それでも文字化けする場合は、Googleスプレッドシートやテキストエディタで開いて確認してください。

### Q4. 書き出し順が思った通りではありません

`Sort by Name` をオンにすると名前順になります。  
順番を安定させたい場合は、オンにすることをおすすめします。

### Q5. 保存先フォルダが存在しない場合はどうなりますか？

修正版では、指定した保存先フォルダが存在しない場合、自動作成を試みます。

例：

```text
//exports/selected_names.csv
```

`exports` フォルダが存在しない場合でも、作成できる環境であれば自動で作られます。

## 12. まとめ：最短の使い方

1. Blenderに `インスタンスヘルパー_選択オブジェクト名CSV書き出し_修正版.py` をインストールする
2. アドオンを有効化する
3. 3Dビューで書き出したいオブジェクトを選択する
4. `N` キーでサイドバーを開く
5. `Instance Helper` タブを開く
6. `Save Path` に保存先を指定する
7. 必要に応じて `Include Header` と `Sort by Name` を設定する
8. `Export Selected Names to CSV` を押す
9. 指定した場所にCSVが作成される

---

## 13. おすすめ設定

通常作業では、次の設定がおすすめです。

```text
Save Path: //exports/selected_object_names.csv
Include Header: オン
Sort by Name: オン
```

この設定にすると、`.blend` ファイルと同じ場所に `exports` フォルダを作り、選択オブジェクト名を確認しやすい名前順のCSVとして保存できます。
