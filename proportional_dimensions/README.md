# Proportional Dimensions

## 概要
指定したX/Y/Z寸法を基準に、選択オブジェクトを縦横比を保ったまま均等スケールするアドオンです。

## 基本情報
- 本体ファイル: `proportional_dimensions.py`
- 表示場所: `View3D > Sidebar(N) > 比率寸法`
- バージョン: `1.0.0`
- 対応Blender目安: `3.6.0` 以降
- カテゴリ: `Object`

## 使う場面
- モデルの比率を崩さずに指定寸法へ合わせたい
- X、Y、Zのどれか1軸を基準に全体を均等スケールしたい
- アクティブだけ、または選択中すべてを同じ操作でサイズ調整したい

## 最短手順
1. サイズ調整したいオブジェクトを選択します。
2. 3D Viewportで `N` キーを押し、`比率寸法` タブを開きます。
3. `現在寸法を読み込み` を押して現在値を確認します。
4. 目標にしたいX/Y/Z寸法を入力します。
5. `X基準`、`Y基準`、`Z基準` のいずれかを押します。

## 主な設定
- `X` / `Y` / `Z`: 目標にする寸法
- `X基準` / `Y基準` / `Z基準`: どの軸の寸法へ合わせるか
- `選択中すべてに適用`: OFFならアクティブのみ、ONなら選択中の編集可能な全オブジェクトへ適用

## 結果
対象オブジェクトのScaleをXYZ同じ倍率で変更し、Dimensionsの比率を維持します。

## 注意点
- 基準軸の現在寸法が0のオブジェクトはスキップされます。
- 複数選択に適用する場合は `選択中すべてに適用` をONにします。
- 寸法単位はBlenderのシーン単位設定に従います。
