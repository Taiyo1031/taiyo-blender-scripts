# GNパラメータCSVエクスポーター 完全使用書

対象ファイル：`GNパラメータCSVエクスポーター_v1_1_修正版.py`  
元ファイル：`ExportModefireParametor.py`  
アドオン名：`GN Parameter CSV Exporter`  
表示場所：`3D Viewport > Sidebar（Nキー）> GN CSV Export`

---

## 1. このアドオンの目的

このアドオンは、Blender内で選択したオブジェクトに付いている **Geometry Nodesモディファイアの入力パラメータ** を、CSVファイルとして書き出すためのツールです。

通常、Geometry Nodesのモディファイア入力値はオブジェクトごとに確認する必要があります。  
このアドオンを使うと、複数オブジェクトのパラメータをまとめて表形式で出力できます。

主な用途は次のとおりです。

- 複数オブジェクトに設定されたGeometry Nodesパラメータの一覧化
- オブジェクトごとのインスタンス設定、サイズ、密度、ON/OFF、素材指定などの確認
- CSVとして書き出し、Excel、Google スプレッドシート、管理表、外部ツールで利用
- 制作中のシーンで、どのオブジェクトにどのGN設定が入っているかをチェック
- 大量のオブジェクトの設定を比較・確認する作業の効率化

出力されるCSVは、基本的に次のような構造になります。

```csv
Object,ParamA,ParamB,ParamC
Cube,1.0,True,Material_A
Sphere,2.5,False,Material_B
```

1行が1オブジェクト、1列が1つのGeometry Nodes入力パラメータです。

---

## 2. このアドオンでできること

このアドオンでは、主に以下の操作ができます。

| 機能 | 内容 |
|---|---|
| 対象モディファイア指定 | 書き出したいGeometry Nodesモディファイア名を指定できます。 |
| アクティブGNモディファイア取得 | アクティブオブジェクト上の最初のGeometry Nodesモディファイア名を自動入力できます。 |
| パラメータ手動追加 | 書き出したいパラメータ名を手入力できます。 |
| GN入力から自動追加 | アクティブオブジェクトのGeometry Nodesグループ入力を読み取り、パラメータ欄へ自動入力できます。 |
| パラメータ削除 | 不要なパラメータ行を個別に削除できます。 |
| 全パラメータクリア | パラメータ一覧をまとめて消去できます。 |
| 選択オブジェクト一括出力 | 選択中の複数オブジェクトを1行ずつCSVに出力できます。 |
| アクティブのみ出力 | 選択全体ではなく、アクティブオブジェクトだけを書き出せます。 |
| ヘッダー行あり/なし | CSVの1行目に列名を出すかどうかを選べます。 |
| Excel対応CSV出力 | UTF-8 BOM付きで保存するため、日本語名もExcelで比較的扱いやすくなっています。 |

## 3. インストール方法

### 3.1 アドオンファイルを用意する

今回使用するファイルは次のファイルです。

```text
GNパラメータCSVエクスポーター_v1_1_修正版.py
```

元の `ExportModefireParametor.py` ではなく、上記の修正版を使用してください。

### 3.2 Blenderにインストールする

1. Blenderを開きます。
2. 上部メニューから `Edit > Preferences` を開きます。
3. 左メニューで `Add-ons` を選びます。
4. `Install...` または `Install from Disk...` を押します。
5. `GNパラメータCSVエクスポーター_v1_1_修正版.py` を選択します。
6. アドオン一覧に表示された `GN Parameter CSV Exporter` にチェックを入れて有効化します。

### 3.3 パネルの表示場所

インストール後、3D Viewportで `Nキー` を押してサイドバーを開きます。

右側のタブに次のタブが追加されます。

```text
GN CSV Export
```

この中に、CSV書き出し用のUIがあります。

## 5. Target Modifier（対象モディファイア）

### 5.1 Modifier Name

```text
Modifier Name
```

ここには、書き出したい **Geometry Nodesモディファイアの名前** を入力します。

例：

```text
GeometryNodes
```

または

```text
Scatter_Setup
```

または

```text
Instance_Controller
```

Blender上のモディファイア名と一致している必要があります。

### 5.2 注意点

Geometry Nodesモディファイア名は、オブジェクトごとに同じ名前にしておくのが理想です。

例えば、複数オブジェクトを一括出力する場合、全オブジェクトに次のように同じ名前のGNモディファイアが付いていると安定します。

```text
GeometryNodes
```

もし選択オブジェクトの中に指定名のモディファイアが存在しない場合、そのオブジェクトの行は空欄として出力されます。

### 5.3 スポイトボタン / Use Active GN Modifier

Modifier Name欄の右側にあるスポイトアイコンのボタンです。

このボタンを押すと、現在のアクティブオブジェクトに付いている最初のGeometry Nodesモディファイア名を自動で入力します。

使う場面：

- モディファイア名を正確に入力するのが面倒な時
- `GeometryNodes.001` など、名前が少し変わっている時
- スペルミスを防ぎたい時

---

## 6. Parameters（CSV列として出力するパラメータ）

ここでは、CSVに出力したいGeometry Nodes入力パラメータを指定します。

CSVでは、最初の列が `Object` になり、その後にここで指定したパラメータが列として並びます。

例：

```text
Object,Size,Density,Material,Use Random
```

---

## 7. Add Param

```text
Add Param
```

パラメータ入力欄を1行追加します。

追加した欄に、書き出したいGeometry Nodes入力名を入力します。

例：

```text
Size
```

```text
Density
```

```text
Instance Collection
```

```text
Random Seed
```

### 7.1 入力する名前について

基本的には、Geometry Nodesモディファイアの入力として表示されている名前を入力します。

ただし、Blender内部では表示名とは別に `identifier` が使われることがあります。  
このアドオンは、次の順番で値を探します。

1. 入力された名前がモディファイア内部キーと一致するか
2. Geometry Nodesの表示名から内部identifierに変換できるか
3. 大文字小文字を無視して一致するキーがあるか

そのため、基本的にはUIに表示されているパラメータ名を入力すれば問題ありません。

---

## 8. Clear

```text
Clear
```

現在追加されているパラメータ行をすべて削除します。

使う場面：

- 別のGeometry Nodesグループを出力したい時
- パラメータ一覧を最初から作り直したい時
- 自動取得をやり直したい時

注意：Clearを押すと、現在のパラメータ一覧はすべて消えます。

---

## 9. Populate from GN Inputs

```text
Populate from GN Inputs (active object)
```

アクティブオブジェクトのGeometry Nodesモディファイアから、グループ入力パラメータを自動で読み取り、Parameters欄に追加します。

### 9.1 基本的な使い方

1. Geometry Nodesモディファイアが付いたオブジェクトを選択します。
2. そのオブジェクトをアクティブにします。
3. `Modifier Name` が正しいことを確認します。
4. `Populate from GN Inputs (active object)` を押します。
5. Parameters欄にGN入力名が自動で並びます。

### 9.2 重要な注意点

このボタンは **アクティブオブジェクト** のGeometry Nodes入力を基準にします。

複数オブジェクトを選択していても、パラメータ一覧を作る時に参照するのはアクティブオブジェクトです。

そのため、最初に「基準にしたいGN設定を持つオブジェクト」をアクティブにしてから押してください。

---

## 10. パラメータ行の右側に表示される値

Parameters欄にパラメータを追加すると、右側に現在のアクティブオブジェクトから読み取った値が表示されます。

例：

```text
Size        2.0
Density     100
Material    Mat_Stone
```

これは確認用の表示です。

ここで値が空欄になる場合、主に次の原因が考えられます。

- パラメータ名が間違っている
- `Modifier Name` が間違っている
- アクティブオブジェクトに対象GNモディファイアがない
- そのパラメータがGeometry Nodesの入力として公開されていない
- 値が未設定または取得できない形式になっている

---

## 11. 個別削除ボタン

修正版では、各パラメータ行の右側に `X` ボタンがあります。

このボタンを押すと、そのパラメータ行だけを削除できます。

使う場面：

- 自動取得した中から不要な列だけ消したい時
- CSVに出したくないパラメータを整理したい時

---

## 12. Export All Selected Objects

```text
Export All Selected Objects
```

このチェックボックスで、CSVに出力する対象を切り替えます。

### ONの場合

選択中のすべてのオブジェクトを出力します。

```text
ON = 選択オブジェクトをすべて出力
```

複数オブジェクトの設定を一覧化したい時に使用します。

### OFFの場合

アクティブオブジェクトだけを出力します。

```text
OFF = アクティブオブジェクトのみ出力
```

1つのオブジェクトだけ確認したい時や、テスト出力したい時に便利です。

---

## 13. Include Header

```text
Include Header
```

CSVの1行目に列名を出すかどうかを設定します。

### ONの場合

次のようにヘッダー行が出力されます。

```csv
Object,Size,Density,Material
Cube,1.0,50,Mat_A
```

通常はON推奨です。  
ExcelやGoogle スプレッドシートで開いた時に見やすくなります。

### OFFの場合

ヘッダー行なしでデータだけ出力されます。

```csv
Cube,1.0,50,Mat_A
```

外部ツールに読み込ませる時など、ヘッダーが不要な場合に使います。

---

## 14. Export CSV

```text
Export CSV
```

指定した内容でCSVを書き出します。

押すと保存先を選ぶ画面が開くので、ファイル名と保存場所を指定します。

推奨ファイル名例：

```text
gn_parameters_export.csv
```

または、用途が分かる名前にします。

```text
school_props_gn_parameters.csv
```

```text
scatter_settings_2026_05_19.csv
```

## 15. 基本ワークフロー

初めて使う場合は、以下の流れが一番安全です。

### 手順

1. Geometry Nodesモディファイアが付いたオブジェクトを選択します。
2. 3D Viewportで `Nキー` を押します。
3. `GN CSV Export` タブを開きます。
4. `Modifier Name` に対象モディファイア名を入力します。
5. 名前が分からない場合はスポイトボタンを押します。
6. `Populate from GN Inputs (active object)` を押します。
7. Parameters欄に入力名が並んだことを確認します。
8. 不要なパラメータがあれば `X` で削除します。
9. 複数オブジェクトを出したい場合は、対象オブジェクトをすべて選択します。
10. `Export All Selected Objects` をONにします。
11. `Include Header` をONにします。
12. `Export CSV` を押します。
13. 保存先とCSV名を指定します。
14. ExcelやGoogle スプレッドシートでCSVを開いて確認します。

## 16. 複数オブジェクトを書き出す時のおすすめ準備

複数オブジェクトを安定して出力するには、次の条件を揃えるのがおすすめです。

- 全オブジェクトに同じ名前のGeometry Nodesモディファイアを付ける
- 同じGeometry Nodesグループ、または同じ入力名を持つグループを使う
- 書き出したい入力はGroup Inputとして公開しておく
- パラメータ名に極端に分かりにくい記号を使わない
- 一度少数のオブジェクトでテスト出力してから、大量出力する

## 18. CSVに出力される値の形式

このアドオンは、Blender内部の値をCSVで扱いやすい文字列に変換します。

| 値の種類 | 出力形式 |
|---|---|
| 数値 | `1.0` のような文字列 |
| Bool | `True` / `False` |
| 文字列 | そのまま |
| Object / Material / Collectionなど | データブロック名 |
| Vector / Color / 配列 | `;` 区切り |
| 値が取得できない場合 | 空欄 |

VectorやColorは、次のように出力されます。

```text
1.0;0.5;0.25
```

## 19. トラブルシューティング

### 19.1 パネルが表示されない

確認点：

- アドオンが有効化されているか
- 3D Viewportで `Nキー` を押しているか
- 右側サイドバーの `GN CSV Export` タブを見ているか
- Blenderを再起動してみる

### 19.2 Populate from GN Inputsで何も追加されない

確認点：

- アクティブオブジェクトにGeometry Nodesモディファイアが付いているか
- `Modifier Name` が正しいか
- Geometry Nodesの入力がGroup Inputとして公開されているか
- 対象モディファイアが本当にGeometry Nodesモディファイアか

### 19.3 CSVの値が空欄になる

確認点：

- パラメータ名が正しいか
- 大文字・小文字・スペースが一致しているか
- 対象オブジェクトに同名のモディファイアがあるか
- そのオブジェクトだけ別のGeometry Nodesグループを使っていないか
- 値が未設定ではないか

### 19.4 一部オブジェクトだけ空欄になる

原因として多いもの：

- そのオブジェクトに指定名のモディファイアがない
- モディファイア名が他のオブジェクトと違う
- Geometry Nodesグループの入力構成が違う
- そのパラメータだけ存在しない

対処：

- 全オブジェクトのモディファイア名を揃える
- 同じGeometry Nodesグループを使用する
- まず1つの代表オブジェクトで正常に出力できるか確認する

### 19.5 日本語が文字化けする

この修正版は `utf-8-sig` でCSVを書き出します。  
Excelで開く場合は比較的文字化けしにくい形式です。

それでも文字化けする場合は、Excel側で次のように読み込んでください。

1. Excelを開く
2. `データ` タブ
3. `テキストまたはCSVから`
4. 文字コードを `UTF-8` にする

## 21. 使用上の注意

このアドオンは、Geometry Nodesの「現在モディファイアに設定されている入力値」をCSVに書き出すツールです。

以下の用途には対応していません。

- CSVからGeometry Nodesパラメータを読み戻す
- CSVを使ってオブジェクトを自動生成する
- ノード内部の全ノード設定を出力する
- Geometry Nodesの評価結果そのものを出力する
- メッシュ頂点情報や属性値を直接CSV化する

必要なのはあくまで、モディファイア入力として公開されているパラメータです。

---

## 22. 最短まとめ

一番基本的な使い方は次のとおりです。

```text
1. GNモディファイア付きオブジェクトを選択
2. Nキーでサイドバーを開く
3. GN CSV Exportタブを開く
4. スポイトボタンで対象モディファイア名を取得
5. Populate from GN Inputsを押す
6. 不要な列をXで削除
7. Export All Selected ObjectsをON
8. Include HeaderをON
9. Export CSVで保存
```

これで、選択オブジェクトごとのGeometry Nodes入力パラメータをCSVとして一覧化できます。
