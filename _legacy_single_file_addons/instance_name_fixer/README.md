# Instance Name Fixer

## 概要
コレクションインスタンスの名前を、参照元コレクション名に揃えるアドオンです。`.001` などの番号サフィックスは保持します。

## 基本情報
- 本体ファイル: `instance_name_fixer.py`
- 表示場所: `View3D > Sidebar > Name Fixer`
- バージョン: `0.1.0`
- 対応Blender目安: `5.1.1` 以降
- カテゴリ: `Object`

## 使う場面
- コレクションインスタンス名がバラバラで管理しにくい
- アウトライナーやCSV出力前に名前を整理したい
- アセット管理上、インスタンス名を元コレクション名に揃えたい

## 最短手順
1. `Name Fixer` パネルを開きます。
2. `スキャン` を押して変更候補を確認します。
3. 候補に問題がなければ `名前変更を実行` を押します。
4. 必要に応じて各 STEP の三角アイコンを押し、セクションを折りたたみます。

## 主な設定
- `スキャン`: 変更対象になるコレクションインスタンスを検出
- `修正候補プレビュー`: 実行前に変更前後の名前を確認
- `名前変更を実行`: プレビュー内容でリネーム
- 各 STEP の三角アイコン: セクションの開閉

## 結果
対象コレクションインスタンスのオブジェクト名を変更します。

## 注意点
- 通常のメッシュオブジェクト名は変更しません。
- インスタンス元コレクションがないものは対象外です。
- 実行前にプレビューを確認してください。
