# Taiyo Blender Extensions Repository

Blender 4.2以降のExtensions用ローカルリポジトリです。

各サブフォルダは `blender_manifest.toml` と `__init__.py` を持つ拡張機能パッケージです。

## パッケージ一覧
- `attribute_csv_exporter`: Attribute CSV Exporter
- `gn_parameter_csv_exporter`: GN Parameter CSV Exporter
- `export_selected_names_csv`: Export Selected Object Names to CSV
- `instance_name_fixer`: Instance Name Fixer
- `proportional_dimensions`: Proportional Dimensions
- `rb_instance_helper`: RB Instance Helper
- `replace_selected_with_active`: Replace Selected with Active
- `uv_channel_placement_tool`: UV Channel Placement Tool
- `viewport_export_selected_meshes`: Viewport Export Selected Meshes
