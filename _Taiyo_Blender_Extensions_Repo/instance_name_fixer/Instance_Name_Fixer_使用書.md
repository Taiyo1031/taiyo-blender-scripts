# Instance Name Fixer 使用書

## 1. 概要

**Instance Name Fixer** は、Blender 内の **コレクションインスタンスのオブジェクト名** を、参照元である **インスタンス元コレクション名** に揃えるためのアドオンです。

コレクションインスタンスを大量に配置したあと、アウトライナー上の名前がインスタンス元のコレクション名と一致していない場合に使用します。

たとえば、現在のオブジェクト名とインスタンス元コレクション名が次のように異なる場合、

```text
現在の名前: Old_Instance
インスタンス元コレクション名: SM_StPr_Gasyou_A
```

このアドオンを使うと、次のように変更できます。

```text
変更後の名前: SM_StPr_Gasyou_A
```

また、Blender が自動で付ける `.001`、`.002` などの数字サフィックスは保持されます。

```text
現在の名前: Old_Instance.001
インスタンス元コレクション名: SM_StPr_Gasyou_A

変更後の名前: SM_StPr_Gasyou_A.001
```

## 3. このアドオンでできること

このアドオンでは、次の作業ができます。

1. シーン内のコレクションインスタンスをスキャンする
2. 名前がインスタンス元コレクション名と一致していないものを検出する
3. 修正候補をプレビューで確認する
4. 問題がなければ一括で名前を変更する
5. `.001`、`.002` などの重複番号を維持したまま修正する

---

## 4. 対象になるオブジェクト

このアドオンが対象にするのは、Blender の **コレクションインスタンス** です。

内部的には、次の条件を満たすオブジェクトが対象になります。

```text
instance_type が COLLECTION
かつ
instance_collection が存在するオブジェクト
```

次のものは対象外です。

- 通常のメッシュオブジェクト
- カメラ
- ライト
- 空オブジェクト
- 通常のコレクションそのもの
- インスタンス元コレクションが存在しないオブジェクト

## 5. インストール方法

### 5.1 ZIP ファイルを展開する

配布された ZIP ファイルを任意の場所に展開します。

ZIP の中には、次の 2 ファイルが入っています。

```text
instance_name_fixer.py
Instance_Name_Fixer_使用書.md
```

### 5.2 Blender にアドオンをインストールする

1. Blender を開きます。
2. 上部メニューから **Edit > Preferences** を開きます。
3. 左側の **Add-ons** を選択します。
4. **Install...** または **Install from Disk** をクリックします。
5. `instance_name_fixer.py` を選択します。
6. アドオン一覧に **Instance Name Fixer** が表示されたら、チェックを入れて有効化します。

---

## 6. アドオンの表示場所

アドオンを有効化すると、3D ビューポートのサイドバーに表示されます。

```text
3D Viewport > Sidebar > Name Fixer
```

サイドバーが表示されていない場合は、3D ビューポート上で `N` キーを押してください。

## 7. メニュー構成

このアドオンの UI は、3 つのステップで構成されています。

```text
Name Fixer
└─ Instance Name Fixer
   ├─ STEP 1 ─ スキャン
   │  └─ スキャン
   ├─ STEP 2 ─ 修正候補プレビュー
   │  ├─ 候補なし
   │  または
   │  └─ 現在の名前 → 変更後の名前
   └─ STEP 3 ─ 実行
      └─ 名前変更を実行
```

## 8. 各メニューの説明

## 8.1 STEP 1 ─ スキャン

### 表示名

```text
STEP 1 ─ スキャン
```

### ボタン

```text
スキャン
```

### 使用目的

シーン内のコレクションインスタンスを調べ、名前の修正が必要なものを検出します。

### チェック内容

スキャン時には、各コレクションインスタンスについて次の 2 つを比較します。

```text
現在のオブジェクト名
```

```text
インスタンス元コレクション名
```

この 2 つのベース名が違う場合、修正候補としてリストに追加されます。

### サフィックスの扱い

`.001`、`.002` などの数字サフィックスは、比較時には一度取り除かれます。

例:

```text
SM_StPr_Gasyou_A.001
```

比較時には次のように扱われます。

```text
SM_StPr_Gasyou_A
```

そのため、ベース名が同じであれば修正対象にはなりません。

### 実行結果

スキャンが完了すると、Blender のステータス表示に次のようなメッセージが出ます。

```text
3 件の修正候補を検出しました
```

---

## 8.2 STEP 2 ─ 修正候補プレビュー

### 表示名

```text
STEP 2 ─ 修正候補プレビュー
```

### 使用目的

スキャンで見つかった修正候補を、実際に名前変更する前に確認します。

表示形式は次のとおりです。

```text
現在の名前 → 変更後の名前
```

### 候補がない場合

修正候補がない場合は、次のように表示されます。

```text
（候補なし）
```

この場合、現在のシーン内には名前修正が必要なコレクションインスタンスがありません。

### 候補がある場合

修正候補がある場合は、次のように表示されます。

```text
Old_Collection_Instance → SM_StPr_Gasyou_A
Old_Collection_Instance.001 → SM_StPr_Gasyou_A.001
```

### 確認するポイント

実行前に、次の点を確認してください。

1. 変更前の名前が意図した対象か
2. 変更後の名前が正しいか
3. `.001` などの番号が保持されているか
4. 変更してはいけないオブジェクトが含まれていないか

---

## 8.3 STEP 3 ─ 実行

### 表示名

```text
STEP 3 ─ 実行
```

### ボタン

```text
名前変更を実行
```

### 使用目的

プレビューに表示された修正候補を実際に適用します。

### ボタンが押せる条件

修正候補が 1 件以上ある場合のみ、**名前変更を実行** ボタンを押せます。

候補がない場合、このボタンは無効になります。

### 実行内容

ボタンを押すと、プレビューに表示されている各オブジェクトの名前が、提案された名前に変更されます。

### 実行後の動作

名前変更が完了すると、修正候補リストは自動でクリアされます。

Blender のステータス表示には、次のようなメッセージが出ます。

```text
3 件の名前を変更しました
```

## 9. 基本的な使い方

### 手順 1: アドオンを有効化する

Blender の Preferences から **Instance Name Fixer** を有効にします。

### 手順 2: 3D ビューポートを開く

3D ビューポートに移動します。

### 手順 3: サイドバーを表示する

`N` キーを押して、サイドバーを表示します。

### 手順 4: Name Fixer タブを開く

サイドバー内の **Name Fixer** タブをクリックします。

### 手順 5: スキャンする

**STEP 1 ─ スキャン** の中にある **スキャン** ボタンを押します。

### 手順 6: 修正候補を確認する

**STEP 2 ─ 修正候補プレビュー** を確認します。

問題がなければ次に進みます。

### 手順 7: 名前変更を実行する

**STEP 3 ─ 実行** の **名前変更を実行** ボタンを押します。

### 手順 8: 結果を確認する

アウトライナーなどで、コレクションインスタンスの名前が正しく変更されたか確認します。

## 10. 名前変更のルール

このアドオンでは、次のルールで名前を変更します。

```text
変更後の名前 = インスタンス元コレクション名 + 既存の数字サフィックス
```

### 例 1: サフィックスなし

```text
現在の名前: Old_Name
インスタンス元コレクション名: Building_A

変更後の名前: Building_A
```

### 例 2: `.001` がある場合

```text
現在の名前: Old_Name.001
インスタンス元コレクション名: Building_A

変更後の名前: Building_A.001
```

### 例 3: `.023` がある場合

```text
現在の名前: Wrong_Name.023
インスタンス元コレクション名: Tree_Set

変更後の名前: Tree_Set.023
```

### 例 4: すでに正しい場合

```text
現在の名前: Tree_Set
インスタンス元コレクション名: Tree_Set

結果: 修正対象にならない
```

### 例 5: サフィックスだけ違う場合

```text
現在の名前: Tree_Set.001
インスタンス元コレクション名: Tree_Set

結果: 修正対象にならない
```

この場合、ベース名は一致しているため、問題なしと判断されます。

## 11. 使用例

### 例: インスタンス名がバラバラになっている場合

#### 修正前

| 現在の名前 | インスタンス元コレクション名 |
|---|---|
| Obj_A | House |
| Temp_Instance.001 | House |
| Tree_Set | Tree_Set |
| Wrong_Name.002 | Rock |

#### スキャン結果

| 現在の名前 | 変更後の名前 |
|---|---|
| Obj_A | House |
| Temp_Instance.001 | House.001 |
| Wrong_Name.002 | Rock.002 |

`Tree_Set` はすでにインスタンス元コレクション名と一致しているため、修正候補には表示されません。

#### 実行後

| 実行前 | 実行後 |
|---|---|
| Obj_A | House |
| Temp_Instance.001 | House.001 |
| Wrong_Name.002 | Rock.002 |

## 12. 使うべき場面

### コレクションインスタンスを大量に使っている場合

複数の建物、植物、小物、背景パーツなどをコレクションインスタンスとして配置している場合、名前が崩れると管理が難しくなります。

このアドオンを使うことで、インスタンス名を元のコレクション名に揃えられます。

### アウトライナーを整理したい場合

アウトライナー上で、どのインスタンスがどのコレクションを参照しているのか分かりやすくなります。

### 他人から受け取った Blender ファイルを整理したい場合

作業者によって命名ルールが違う場合でも、インスタンス元コレクション名を基準に整理できます。

### アセット管理を安定させたい場合

名前を統一することで、後工程の確認、書き出し、チェック作業がしやすくなります。

## 13. 注意点

### 13.1 通常のオブジェクト名は変更されない

このアドオンが対象にするのは、コレクションインスタンスだけです。

次のようなものは変更されません。

- 通常のメッシュオブジェクト
- カメラ
- ライト
- 空オブジェクト
- 通常のコレクション

### 13.2 インスタンス元コレクションがないものは対象外

コレクションインスタンスであっても、インスタンス元コレクションが存在しない場合は対象外です。

### 13.3 実行前に必ずプレビューを確認する

名前変更は一括で実行されます。

実行前には、必ず **STEP 2 ─ 修正候補プレビュー** を確認してください。

### 13.4 Undo に対応している

このアドオンの名前変更実行処理は、Blender の Undo に対応しています。

実行後に戻したい場合は、通常の Undo 操作を使えます。

```text
Ctrl + Z
```

## 14. よくある質問

### Q1. スキャンしても「候補なし」と表示されます

シーン内に修正が必要なコレクションインスタンスがない状態です。

次の可能性があります。

1. すでに名前が正しい
2. コレクションインスタンスが存在しない
3. 対象が通常のメッシュオブジェクトである
4. インスタンス元コレクションが設定されていない

### Q2. `.001` が消えません

仕様です。

このアドオンは、Blender が付ける `.001`、`.002` などのサフィックスを保持します。

これは、同名オブジェクトの衝突を避けるためです。

### Q3. コレクション名そのものは変更されますか？

変更されません。

このアドオンが変更するのは、コレクションインスタンスの **オブジェクト名** です。

インスタンス元のコレクション名は変更されません。

### Q4. メッシュオブジェクトの名前も直せますか？

このアドオンでは直せません。

対象はコレクションインスタンスのみです。

### Q5. 実行後に候補リストが消えました

正常な動作です。

名前変更を実行すると、修正候補リストは自動でクリアされます。

もう一度確認したい場合は、再度 **スキャン** を押してください。

## 15. トラブルシューティング

### 15.1 Name Fixer タブが表示されない

次を確認してください。

1. アドオンが有効化されているか
2. 3D ビューポートを開いているか
3. サイドバーが表示されているか
4. `N` キーでサイドバーを表示したか

### 15.2 スキャンボタンを押しても候補が出ない

次を確認してください。

1. シーン内にコレクションインスタンスがあるか
2. 対象のオブジェクトが本当にコレクションインスタンスか
3. インスタンス元コレクションが設定されているか
4. すでに名前が正しくなっていないか

### 15.3 間違って名前変更してしまった

Blender の Undo を使って戻してください。

```text
Ctrl + Z
```

## 16. 推奨ワークフロー

実務で使う場合は、次の流れがおすすめです。

1. 作業ファイルを保存する
2. Instance Name Fixer を開く
3. **スキャン** を押す
4. プレビューを確認する
5. 問題がなければ **名前変更を実行** を押す
6. アウトライナーで結果を確認する
7. ファイルを保存する

---

## 17. まとめ

**Instance Name Fixer** は、コレクションインスタンスの名前を整理するためのシンプルなアドオンです。

主な機能は次の 3 つです。

```text
スキャン
修正候補プレビュー
名前変更を実行
```

このアドオンを使うことで、コレクションインスタンスの名前をインスタンス元コレクション名に揃え、アウトライナーやアセット管理を分かりやすく保つことができます。
